# Уровни Агентности


| Уровень управления    | Как ты управляешь                   | Что агент делает                                                          | Примеры “понятными тебе продуктами”                                                                 |
| --------------------- | ----------------------------------- | ------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| **L1 — Оператор**     | Ты полностью ведёшь процесс         | Только отвечает/объясняет по запросу                                      | **ChatGPT (обычный чат в браузере)**                                                                |
|                       |                                     |                                                                           |                                                                                                     |
| **L2 — Коллаборатор** | Ты задаёшь рамки, агент делает шаги | Выполняет работу, но в узком сценарии и под твоими командами              | **web-Codex: “не меняй код, давай подумаем”** (анализ/план/обсуждение без правок)                   |
|                       |                                     |                                                                           |                                                                                                     |
| **L3 — Консультант**  | Уточнения редко                     | Сам ведёт многошаговую задачу и приносит результат                        | **Deep Research (в браузере)**                                                                      |
|                       |                                     |                                                                           |                                                                                                     |
| **L4 — Утверждающий** | Ты даёшь OK только когда нужно      | Сам работает и может вносить изменения; ты вмешиваешься только при рисках | **web-Codex: “сделай то-то и то-то”** (ожидаешь, что он сам внесёт правки в ветке и доведёт задачу) |
|                       |                                     |                                                                           |                                                                                                     |
| **L5 — Наблюдатель**  | Ты только смотришь                  | Полный автопилот без участия человека                                     | *(из твоих примеров — нет прямого соответствия)*                                                    |
