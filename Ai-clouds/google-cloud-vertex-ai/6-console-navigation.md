# 🗺 Шпаргалка по интерфейсу Google Cloud Console

Google Cloud Console может быть запутанной. Вот где искать самое важное для работы с Vertex AI:

* **Узнать настоящий Project ID:**
  Нажмите на название вашего проекта в самой верхней синей панели (слева от строки поиска). В открывшемся окне в таблице будет колонка "ID" — именно это значение нужно подставлять в Python-скрипт.
* **Включить Vertex AI:**
  Главная строка поиска (наверху по центру) -> Введите `Vertex AI API` -> Нажмите первую ссылку со значком шестеренки -> Нажмите `ENABLE`.
* **Привязать карту/промо-кредиты:**
  Боковое меню (гамбургер слева вверху) -> **Billing** -> *Link a billing account*.
* **Создать сервисный аккаунт (для продакшна):**
  Боковое меню -> **IAM & Admin** -> **Service Accounts**.
* **Отключить запрет на скачивание ключей:**
  Боковое меню -> **IAM & Admin** -> **Organization Policies** -> В поиске фильтров вбейте `iam.disableServiceAccountKeyCreation` -> Измените настройку на `Off`.
