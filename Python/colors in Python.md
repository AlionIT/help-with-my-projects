
# Основные цвета текста в Python:
    Чёрный: \033[30m
    Красный: \033[31m
    Зелёный: \033[32m
    Жёлтый: \033[33m
    Синий: \033[34m
    Пурпурный: \033[35m
    Голубой: \033[36m
    Белый: \033[37m
    Яркие (bold) версии цветов:
    Яркий чёрный: \033[90m
    Яркий красный: \033[91m
    Яркий зелёный: \033[92m
    Яркий жёлтый: \033[93m
    Яркий синий: \033[94m
    Яркий пурпурный: \033[95m
    Яркий голубой: \033[96m
    Яркий белый: \033[97m


    Сброс стилей:
    Сброс всех настроек: \033[0m
